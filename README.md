# ♠ PkCardVision

> AI-powered Computer Vision Platform for Playing Card Recognition

## 🎯 Vision

PkCardVision is a real-time computer vision platform designed to recognize and analyze playing cards using deep learning.

## 🚀 Features

- Real-time screen capture
- Playing card detection
- Object tracking
- Session recording
- Statistics
- Modern desktop interface

## 🛠 Stack

- Python
- OpenCV
- Ultralytics YOLO
- PyTorch
- PySide6
